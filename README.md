# Chess AI Project

Un'IA scacchistica basata su Deep Learning ispirata ad AlphaZero.

## Installazione

1. Clona il repository
2. Installa i requisiti:
   ```
   pip install -r requirements.txt
   ```

## Utilizzo

Per addestrare il modello:
```
python src/training/trainer.py
```
